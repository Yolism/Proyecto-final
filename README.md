# Proyecto-final 
juego_🐍 

Reaizado por Yolanda Sinchi Mendez

Domingo 28 de Junio 2026

Objetivo: Desarrollar un sistema de juego clásico aplicando una separación de arquitectura en tres capas independientes, en las cuales se tenga menor dificultad al ejecutar el programa encontrar los bugs de forma mas eficiente en ahorro de tiempo.

Funcionalidades: Menú principal interactivo, registro persistente del último score en disco, contador de tiempo real, sistema de colisiones con bordes/cuerpo y frutas aleatorias que incrementan la dificultad. Usando la libreria de Pygame.
